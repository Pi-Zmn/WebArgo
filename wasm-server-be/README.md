## Description

This application is build with the [Nest](https://github.com/nestjs/nest) TypeScript framework. This directory holds the Backend implementation of the project.

## Installation

```bash
$ npm install
# or 
$ npm clean install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
